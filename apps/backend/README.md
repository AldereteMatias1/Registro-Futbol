# Registro Futbol Amigos

API para registrar jugadores, partidos, participaciones, goles y resultados.

## Configuracion

Variables requeridas:

- `DATABASE_URL`
- `ADMIN_USER`
- `ADMIN_PASS`

## Comandos

```bash
npm install
npm run db:migrate
npm run start:dev
```

Documentacion OpenAPI en `/docs`.
